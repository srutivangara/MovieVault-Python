# 🎬 MovieVault

A Personal OTT Watchlist Manager built using Python and Streamlit.

## ✨ Features

- Add Movies
- Search Movies
- Edit Movie
- Delete Movie
- Rating
- Reviews
- Dashboard
- OTT Platform
- Genre
- Watch Status

## 🛠 Tech Stack

- Python
- Streamlit
- JSON
